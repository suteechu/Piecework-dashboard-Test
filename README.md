# Piece Work Dashboard (Streamlit)

## วิธีใช้งาน (VS Code)
1. เปิดโฟลเดอร์นี้ด้วย VS Code
2. เปิด Terminal ใน VS Code แล้วรันคำสั่งต่อไปนี้:
```bash
python -m venv .venv
# Windows: .venv\Scripts\activate
# macOS/Linux: source .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```
3. วางไฟล์ Excel: data/ค่าแรงงาน Piece work.xlsx
4. เปิดลิงก์ http://localhost:8501
